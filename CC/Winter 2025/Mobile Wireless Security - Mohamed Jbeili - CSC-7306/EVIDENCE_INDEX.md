## Screenshots

_Screenshots directory reserved for additional evidence images (e.g., configuration captures, test output, architecture diagrams) that supplement the narrative portfolio documents. Populate using the recommended naming conventions below._

### Naming Conventions

- Week/topic pattern: `wkNN_<topic>_<index>.png` (e.g., `wk02_wpa2_config_1.png`)
- Generic evidence: `Screenshot<index>_<short-desc>.png` (e.g., `Screenshot3_MAC_ACL.png`)
- All lowercase, no spaces, use dashes or underscores
