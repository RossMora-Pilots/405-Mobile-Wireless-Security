# Security Policy

## Scope

This repository is an educational portfolio containing documentation, lab
reports, and configuration screenshots from a Mobile & Wireless Security
course. It does not contain production infrastructure, live credentials, or
executable services.

## Reporting a Vulnerability

If you discover a security issue in this repository (e.g., leaked credentials,
exposed PII, or a vulnerability in the automation scripts), please report it
responsibly:

1. **Do not** open a public issue
2. Email **ross.moravec@outlook.com** with a description of the finding
3. Allow 48 hours for an initial response

## What to Report

- Accidentally committed secrets or credentials
- Personally identifiable information (PII) beyond the author's name
- Student identifier (e.g., A00XXXXXX) exposed in any file
- Vulnerabilities in CI/CD workflows or automation scripts
- Insecure file permissions or configurations

## Acknowledgments

Valid reports will be acknowledged in this file. Thank you for helping keep
this portfolio secure.
